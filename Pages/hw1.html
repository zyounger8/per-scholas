Meet Zoraida Younger

Social Worker Gone Tech

Hi! I am Zoraida Younger, but you can call me Zee. I am natural born empath. This led me to the field of Social Work, making impacts by helping build programs and applications from a human services perspective.
Now I desire to create change through my code.
Growing up I did not see women of color that worked in the tech industry. I wish I did because I beleive that I would have realized my love for coding sooner.
Today I've met many women in the tech field and it inspires me to know that if don't give up I can be a woman in tech too. Below is an image of a black woman coding. I look to this as inspiration to keep going.

What I am interested in learning:

  • HTML
  • CSS
  • Javascript
  • React

Thank you for taking some time to learn about me as I start my journey to becoming a Front End Web Developer!
